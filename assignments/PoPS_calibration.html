Run each block in the provided R scripts step by step and see the associated instructions as comments in the code.

Calibration

Run the calibration code to derive parameters for spore rate and distance for the PoPs model.
  • What are the parameters?
  • How many times were the spore rate and distance parameters used out of the total number of calibration iterations?
  • What does the distribution of the two parameters look like?
Optional:
  • Try different numbers of iterations. Are derived parameters different?
  • Try turning management on (line 22). Are derived parameters different?

Validation

Use the parameters derived from the calibration code to run the PoPS model and validate the outputs using the validation code.
  • What was the average disagreement and landscape similarity?
Optional:
  • Try with management on (line 24). How are the total disagreement and landscape similarity affected?
  • Try with different spore rate and distance parameters. How are total disagreement and landscape similarity affected?
  • Try with different conditions. How are total disagreement and landscape similarity affected?
    • With mortality on - Set mortality to "TRUE" and you can change the conditions for time lag and mortality rate (lines 28-30)
    • With wind direction on - Can set wind to "N", "S", "W", "E". Higher kappa values make the wind direction stronger (lines 33-34).
