Assignment part 4.B

Perform simulations of pathogen spread for various scenarios using r.pops.spread and calibrate the spread model using rpops:
  • r.pops.spread tutorial for Sudden Oak Death (SOD) disease, alternatively run the tutorial in a Jupyter Notebook using Binder
  • PoPS model calibration
  • Optional: Explore Spatiotemporal Epidemiological Modeler (STEM) for COVID-19. There are downloadable sample runs available for various epidemics.
